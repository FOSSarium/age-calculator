# Age Calculator

Calculate your exact age in years, months, and days with precision.

## ✨ Features

- 📅 **Precise Calculation** - Shows age in years, months, and days
- 🎂 **Birthday Countdown** - See days until your next birthday
- 📱 **Responsive Design** - Works on desktop, tablet, and mobile
- 🌓 **Dark/Light Mode** - Choose your preferred theme
- ⚡ **Instant Results** - Real-time calculation as you type

## 🚀 Usage

1. Select your birth date
2. (Optional) Select the target date (defaults to today)
3. View your exact age instantly!

## 🛠️ Built With

- HTML5
- CSS3 (Custom Properties, Flexbox, Grid)
- Vanilla JavaScript
